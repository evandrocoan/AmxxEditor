


# AmxxEditor

By default,this version do not create the `AmxxEditor` menu, because not everybody want that menu showing up.

You can add the menu going on the Sublime Text menu `Preferences -> Browse Packages...` and creating the
file `Main.sublime-menu` on the `User/` directory or any subfolder/directory inside it:

    [
        {
            "caption": "AmxxEditor",
            "id": "AmxxEditor",
            "children":
            [
                {
                    "command": "new_amxx_include",
                    "caption": "New Include"
                },
                {
                    "command": "new_amxx_plugin",
                    "caption": "New Plug-In"
                },
                {
                    "command": "open_file",
                    "args": {"file": "${packages}/User/AmxxEditor/AmxxEditor.sh"},
                    "caption": "Configure Linux/Cygwin Compiler"
                },
                {
                    "command": "open_file",
                    "args": {"file": "${packages}/User/AmxxEditor/AmxxEditor.bat"},
                    "caption": "Configure Windows/Bat Compiler"
                },
                { "caption": "-" },
                {
                    "command": "open_file",
                    "args": { "file": "${packages}/User/AmxxEditorConsole.sublime-settings" },
                    "caption": "Edit AmxxEditor Console Settings"
                },
                {
                    "command": "open_file",
                    "args": { "file": "${packages}/User/AmxxEditor.sublime-settings" },
                    "caption": "Edit AmxxEditor Autocompletion Settings"
                },
            ]
        },
    ]
